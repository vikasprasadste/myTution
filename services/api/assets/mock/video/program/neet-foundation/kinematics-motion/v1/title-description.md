# Motion in a Straight Line: displacement, velocity, acceleration

Short concept video for Class 11 medical foundation learners.

## Description

This lesson introduces one-dimensional motion using three linked ideas: displacement tells where an object ends relative to where it began, velocity explains how fast displacement changes, and acceleration explains how fast velocity changes. Learners also review units, sign convention, and what slopes and areas mean on basic motion graphs.

## Learning Outcomes

- Differentiate distance and displacement.
- Read average speed and average velocity from a short scenario.
- Identify acceleration from a changing velocity.
- Interpret slope on position-time and velocity-time graphs.
- Use standard SI units for kinematics quantities.
